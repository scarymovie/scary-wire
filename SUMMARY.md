# WireGuard VPN - Сводка проекта

## Описание
Полный набор bash-скриптов для автоматического развертывания и управления WireGuard VPN сервером.

## Основные возможности

### ✅ Автоматическая установка
- Установка WireGuard на Ubuntu/Debian/CentOS/RHEL
- Настройка IP forwarding
- Конфигурация firewall (UFW/firewalld)
- Генерация ключей сервера и клиентов
- Создание 3 клиентов по умолчанию

### ✅ Управление клиентами
- Добавление новых клиентов
- Удаление клиентов
- Список всех клиентов
- Генерация QR кодов для мобильных устройств
- Настройка split-tunnel

### ✅ Мониторинг и диагностика
- Мониторинг подключений в реальном времени
- Проверка статуса системы
- Генерация отчетов об использовании
- Тестирование соединения
- Просмотр логов

### ✅ Резервное копирование
- Создание резервных копий
- Восстановление из резервных копий
- Автоматическое сохранение перед удалением

### ✅ Дополнительные функции
- Обновление IP адреса сервера
- Интерактивное меню
- Полная документация на русском

## Быстрый старт

### Установка
```bash
sudo bash setup-wireguard.sh
```

### Интерактивное меню
```bash
sudo bash quick-start.sh
```

### Добавить клиента
```bash
sudo bash add-client.sh laptop 10.0.0.5
```

### Показать QR код
```bash
sudo bash show-qr.sh laptop
```

### Мониторинг
```bash
sudo bash monitor.sh
```

## Технические детали

### Сеть
- Сервер: 10.0.0.1/24
- Клиенты: 10.0.0.2 - 10.0.0.254
- Порт: 51820/udp
- DNS: 1.1.1.1, 8.8.8.8

### Криптография
- Шифрование: ChaCha20
- Аутентификация: Poly1305
- Обмен ключами: Curve25519
- Хеширование: BLAKE2s

### Требования
- Linux сервер (Ubuntu/Debian/CentOS/RHEL)
- Root доступ
- Открытый UDP порт 51820
- Публичный IP адрес (желательно статический)

## Безопасность
- Все ключи хранятся с правами 600
- Автоматическая настройка firewall
- NAT masquerading для безопасной маршрутизации
- Perfect Forward Secrecy
- Резервное копирование перед критическими операциями

## Производительность
- Пропускная способность: до 1 Гбит/с
- Задержка: < 1ms overhead
- Низкое потребление CPU
- Поддержка до 253 клиентов
