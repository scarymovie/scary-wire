# Changelog

## [1.1.0] - 2026-05-21

### Добавлено
- Автоматический обход .ru и .рф доменов через `setup-ru-bypass.sh`
- Загрузка списка российских IP подсетей из RIPE
- Автоматическое управление маршрутами через PostUp/PostDown
- Интеграция в основной установочный скрипт
- Документация RU_BYPASS.md
- Пункт в интерактивном меню для настройки обхода

### Изменено
- Обновлен setup-wireguard.sh с вопросом про обход
- Обновлен quick-start.sh с новым пунктом меню
- Обновлен help.sh с командой setup-ru-bypass.sh
- Обновлен README.md со ссылкой на RU_BYPASS.md

## [1.0.0] - 2026-05-21

### Добавлено
- Основной скрипт установки `setup-wireguard.sh`
- Управление клиентами: `add-client.sh`, `remove-client.sh`, `list-clients.sh`
- Отображение QR кодов: `show-qr.sh`
- Мониторинг в реальном времени: `monitor.sh`
- Резервное копирование и восстановление: `backup.sh`, `restore.sh`
- Проверка статуса системы: `check-status.sh`
- Обновление endpoint: `update-endpoint.sh`
- Генерация отчетов: `generate-report.sh`
- Настройка split-tunnel: `setup-split-tunnel.sh`
- Тестирование соединения: `test-connection.sh`
- Удаление системы: `uninstall.sh`
- Интерактивное меню: `quick-start.sh`
- Полная документация на русском языке

### Особенности
- Автоматическая установка и настройка
- Поддержка Ubuntu, Debian, CentOS, RHEL
- Автоматическая настройка firewall (UFW/firewalld)
- Генерация QR кодов для мобильных устройств
- Мониторинг подключений в реальном времени
- Система резервного копирования
- Split-tunnel поддержка
